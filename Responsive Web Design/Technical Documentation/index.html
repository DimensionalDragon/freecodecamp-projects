Javascript Array Functions
Push Pop Filter Map Reduce
Push
arr.push(value)

Used to insert an element to the back of an array

This method will not create a new array and will change the existing array

Parameters used:

  • value: the value/element to be inserted
Pop
arr.pop()

Used to remove the last element of an array

This method will not create a new array and will change the existing array

Filter
arr.filter(function(element, index, array))

Used to filter elements on an array based on certain criteria

Every element will be checked using a function, elements that returns false will be deleted

This method will return the filtered array, so be sure to store it in a variable

Parameters used:

  • element: represents each element in an array, will be used inside the filter function
  • index (optional): represents the index of the element, use this in case you would like to use the index in the filter function
  • array (optional): represents the whole array, use this in case you would like to use the array in the filter function
Map
arr.map(function(element, index, array))

Used to map each element of an array to different type/value

Every element will be mapped using a function, elements will be mapped to the return value of that function

This method will return the mapped array, so be sure to store it in a variable

Parameters used:

  • element: represents each element in an array, will be used inside the mapping function
  • index (optional): represents the index of the element, use this in case you would like to use the index in the mapping function
  • array (optional): represents the whole array, use this in case you would like to use the array in the mapping function
Reduce
arr.reduce(function(accumulator, element, index, array), initial)

Used to reduce an array to 1 element

Usually implemented in finding sum, average, and other collective operations

This method will return the final value, so be sure to store it in a variable

Parameters used:

  • accumulator: A variable that will accumulate operations for every element
  • element: represents each element in an array, will be used inside the accumulating function
  • index (optional): represents the index of the element, use this in case you would like to use the index in the accumulating function
  • array (optional): represents the whole array, use this in case you would like to use the array in the accumulating function
  • initial (optional): sets a default value for accumulator, defaults to the first value of array if not specified
